Class: Mat2

Mat2()

Mat2 is a 2x2 linear algebra matrix

Elements are stored in 'column major' layout [Cite.2], i.e. for matrix M with math convention M_rc (r=row,c=column):

   this.array = [ M_00, M_10,    // first column
                  M_01, M_11 ];  // second column

Details:

  • Note, column major order is consistent with OpenGL and GLSL [Cite.3].
  • For efficiency and GLSL compatibility we use a TypedArray [Cite.1].
